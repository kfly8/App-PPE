# NAME

ppe - Prettify Perl Error messages

# SYNOPSIS

    % perl ~/foo.pl 2>&1 | ppe
    foo.pl:5: [WARN] (W) Use of uninitialized value in warn

# DESCRIPTION

[ppe](https://metacpan.org/pod/ppe) is a prettifier for Perl error messages.
